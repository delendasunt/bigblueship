HWL Importable Map Converter

The Big Blue Ship is a tool to convert Homeworlds gamestates from intuitive plaintext to specialized markup which can be imported into the Homeworlds Live Sandbox. Being able to conveniently play both sides of an ongoing game makes position analysis much easier if you aren't so good at visualizing the gamestate after several hypothetical moves!

The tool is simple to use, but the syntax requirements are moderately precise:

In the freeform text entry field below: type each system as follows, one system per line with no blank lines, without quotations: "system name, enemy ships, star(s), ally ships". Single spaces may be added wherever you wish, but double spaces will fail to parse. For example, a typical starting homeworld might look like "delendasunt, , b1r2, g3". Ordering the fields this way makes it easy to transcribe a system simply by reading it from left to right.

Alternatively, you can input data points singly using the table further down the page. The syntax remains the same, but omit the commas: they will be added automatically. Liberal use of the tab key, and greater ease of error correction, can make this method somewhat faster than freeform typing.

Note that the first two systems will always parse as the south and north homeworlds, respectively. Also, primitive error detection is available, but this is mostly focused on scenarios which are physically impossible, not with enforcing the rules of the game. If you want a system with 10 stars, it's not gonna stop you. Error detection will be expanded soon.

The tool should work repeatedly, but if it begins to behave strangely, try reloading the page: the odd variable has been known to get stuck every now and then. If you encounter any persistent bugs, desire more thorough documentation, or feel like cleaning up some spaghetti code, check out this project's GitHub repository, or message @forkofnature on the Homeworlds Discord server.

Enjoy!
